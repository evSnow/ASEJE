# ASEJE Sidebar Guide

The ASEJE sidebar provides quick access to beginner tools, references, and debugging helpers.

## Features
- Quick navigation to reference materials
- Search functionality for tools and documentation
- Integration with hover and debugger features

## Usage
1. Open the ASEJE sidebar from the VS Code activity bar.
2. Use the search box to find references or tools.
3. Select items to trigger commands or open documentation.
