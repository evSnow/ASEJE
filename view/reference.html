Python Escape Sequences

Escape sequences are special character combinations used to represent "untypeable" characters like new lines or quotes inside a string.

Character Name Escape Sequence
Newline \n
Horizontal Tab \t
Backslash \\
Single Quote \'
Double Quote \"
