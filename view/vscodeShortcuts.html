VS Code Shortcuts Guide

This page gives new users a quick list of helpful Visual Studio Code shortcuts. Learning these shortcuts can make coding faster and easier.

Shortcut What it does
Ctrl + S Save the current file
Ctrl + Shift + P Open the Command Palette
Ctrl + / Comment or uncomment the selected line
Ctrl + ` Open or close the integrated terminal
Ctrl + F Find text in the current file
Ctrl + H Find and replace text
Ctrl + P Quickly open files by name
Ctrl + , Open Settings
F5 Start debugging
Shift + F5 Stop debugging
