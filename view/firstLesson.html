Module 1: Basics

Activity Bar Overview

The Activity Bar is your command center in VS Code. Understanding these icons will help you navigate your project and debug code like a pro.

What is the Activity Bar?

Located at the edge of your window, this toolbar holds the most frequently used tools. Whether you are searching for code or managing your project files, this is where you start.

💡 ASEJE Tip: Our extension simplifies this view to focus only on what you need for your current lesson.

Core Navigation Icons

  • Explorer: Manage your files and project folders.
  • Search: Find specific text across all your files.
  • Source Control: Connect your code to GitHub.
  • Extensions: Add new powers to your editor.

Run and Debug

This is where the magic happens. Use this icon to find mistakes (bugs) in your code.

Standard Controls:

  • Start / Continue: Kick off your program.
  • Step Over: Move to the next line of code.
  • Step Into: Look inside a specific function.
  • Stop: End the program immediately.

Your First Debug Session

  1. Open your Python or C++ file.
  2. Click Run and Debug in the sidebar.
  3. Set a Breakpoint by clicking next to a line number.
  4. Press the green arrow to begin your journey!
