Lesson One: Activity Bar Overview

In this section we will explain the basic of the activity bar and function of each icon
this is important because these icons will help complete the important task and improve work flow
This is a simple explanation more detailed one will be given in another lesson.

What is it

The activity bar is a icon on either left, right or top.
The icons will have functions that all developers will need when coding.

Explorer

This function allows users to be able to open a folder and manage the content of the directory.
This can include adding new files and directory along with changing their names.

Source code

This icon's main function is to control the git version for github allowing for quick push and pull along with staging.

Run and debug

This function can be used to debug code and create breakpoints and run the code
this debugging can be done using other extension as well

Extension

Finally is the extension function which open the marketplace were other users can publish their own extension along with any needed infomation or linked to their githubs
