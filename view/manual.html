📘 ASEJE Manual

This manual explains every major feature included in the ASEJE extension.

1. What ASEJE Is

ASEJE is a beginner-focused VS Code extension designed to help new developers better understand coding, debugging, and editor tools in a simpler and more guided way.

2. Debugger

The debugger is one of the main features of ASEJE. It helps users understand how their code runs step by step instead of guessing why something failed.

  • Step In: Moves into a function to show what happens inside it.
  • Step Out: Exits the current function and returns to the higher stack level.
  • Continue: Runs the program until the next breakpoint.
  • Breakpoints: Pause execution at important lines to inspect behavior.

3. Audio Cues

ASEJE supports audio feedback during debugging. Sound can play when a breakpoint is hit or when a program ends.

  • Helps reinforce learning through sound
  • Can be enabled or disabled by the user
  • Can use custom audio files
  • Supported formats include MP3, WAV, and M4A

4. Hover Explanations

The hover feature helps explain code directly inside VS Code. This is especially useful for new developers who may not understand what a loop, print statement, or variable is doing.

  • Provides beginner-friendly explanations
  • Helps explain Python syntax and behavior
  • Can make errors easier to understand
  • Can be turned on or off

5. Sidebar

The ASEJE sidebar provides a structured place for users to access extension features. It is designed to make navigation easier and reduce confusion.

  • Organized menus and sections
  • Quick access to extension tools
  • Supports notes and reference-style content
  • Can update based on user actions

6. Starter Templates

ASEJE includes starter templates to help beginners begin projects faster and better understand how files are structured.

  • Templates for Python
  • Templates for HTML
  • Templates for JavaScript
  • Templates for C

7. Walkthroughs

The walkthrough system helps users learn the extension step by step. It introduces important tools and explains how to use core features in Visual Studio Code.

8. Future Improvements

  • AI-powered code explanations
  • More language support beyond Python
  • Performance insights and optimization help
  • Custom learning modes like Beginner, Intermediate, and Advanced
