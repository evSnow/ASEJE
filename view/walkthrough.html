Welcome to ASEJE

We’re excited to have you here! ASEJE is your beginner-friendly assistant for Visual Studio Code(Extension) designed to make learning, coding, and personalizing your workspace simple and fun.

Let’s take a quick tour!

What is ASEJE?

ASEJE stands for Adaptive Starter Extension for Junior Engineers. It helps new programmers feel comfortable in VS Code by simplifying the interface and providing quick-start coding templates for popular languages.

Key Features

  • Beginner Mode: Simplifies the VS Code UI with one click.
  • Starter Templates: Instantly create “Hello World” projects in Python, HTML, and more.
  • Text Settings: Customize font size, theme, and layout visually.
  • Guided Walkthroughs: Learn all the vscode essential features and coding fundamentals directly inside VS Code.

Getting Started

To begin exploring ASEJE:

  1. Press Ctrl + Shift + P (or Cmd + Shift + P on Mac).
  2. Type ASEJE to see available commands.
  3. Start with “Toggle Beginner Mode (ASEJE)” to simplify your editor.
Tip: You can change font size in ASEJE’s Text Settings panel!
